# DAOT-Titan-Practice

## Simple titan killing practice map
- [Map download](https://github.com/SoesDesu/DAOT-Titan-Practice/releases)
- [Danny's AOT mod](https://modrinth.com/mod/dannys-aot) (required)

# Features:
- Timer to track your progress
- Customizable gamerules
- Customize the amount of titans
# WIPs:
- Personal best time leaderboard
- Multiple maps to play on
- Customize and choose your loadout
